## 仿外卖app

### 技术栈

​	vue全家桶（vue、vue-router、vuex）

​	swiper滑动插件、vue-lazyload懒加载

​	使用了mint-ui部分组件、stylus预加载

​	mock模拟数据等

​	node做后台
